# 🛠️ このリポジトリの直し方・ふやし方ガイド

これは「とみよこゲームランド」を **家族みんな**、そして **AIコーディングツール（Codex など）** が
安全に直したり、ゲームを増やしたりするための説明書です。
人間が読んでもAIが読んでも分かるように書いてあります。

公開ページ: **https://shoulang0729.github.io/kids-game/**

---

## ⭐ いちばん大事なルール

1. **`main` ブランチに直接コミット／push しない。**
   かならず新しいブランチを作って、そこで作業する。
2. 作業がおわったら **プルリクエスト（PR）** を作る。
3. **公開（`main` へのマージ）は、おうちの人が中身を確認してからOKする。**
   → こうすれば、まちがえて直しても本番の公開サイトはこわれません。
4. PRを作るとCodeRabbit（レビューAI）が自動でチェックしてくれます。指摘が出たら直しましょう。

> AIエージェントへ: このリポジトリでは `main` への直接 push は禁止です。
> 変更は必ずブランチを切って PR を作成してください。マージは人間（リポジトリ所有者）が行います。

## 🗺️ リポジトリの地図（構成）

```
kids-game/
├── index.html              ← 入り口ページ（ゲーム一覧）
├── games/
│   ├── base-defense/index.html   ← 地獄基地防衛（こうちゃん）
│   ├── snake/index.html          ← へびゲーム（としくん）
│   ├── youchan-battle/index.html ← ようちゃんバトル（ようちゃん）
│   ├── planet-smash/index.html   ← 惑星ぶっこわし3D（ようちゃん）
│   └── cooking/index.html        ← 料理ゲーム（こうちゃん）
├── wiki/index.html         ← せつめいしょ（あそびかたガイド）
├── README.md               ← プロジェクトの説明
└── CONTRIBUTING.md         ← このファイル
```

- ゲームは **1つにつき1フォルダ**、中に `index.html` を置きます。
- 各ゲームは **HTML/CSS/JavaScript だけ** で動く1ファイル完結が基本です（ビルド不要）。

---

## ➕ 新しいゲームを追加する手順

1. `games/` の中にフォルダを作る（英語の名前。例：`games/maze/`）。
2. その中に `index.html` を作る（中身は1ファイルのゲーム）。
   - `<head>` に必ずこれを入れる（スマホ対応）:
     ```html
     <meta name="viewport" content="width=device-width, initial-scale=1.0">
     ```
3. 入り口ページ `index.html` の `games` リストに1ブロック足す:
   ```js
   {
     title: "めいろ",
     emoji: "🌀",
     desc: "ゴールをめざそう！（だれだれ作）",
     url: "games/maze/index.html",
     ready: true
   }
   ```
4. （あれば）`wiki/index.html` に遊び方セクションを追加。
5. `README.md` のゲーム一覧テーブルと構成ツリーを更新。

## 🔧 ゲームを直す手順

1. 直したいゲームの `games/<名前>/index.html` を開く。
2. 直す。
3. ブラウザで開いて動くか確認する（できれば**スマホ幅**でも確認）。
4. ブランチを作って PR にする。

---

## 📱 スマホで綺麗に見せるコツ

- 文字やボタンが大きすぎたら `@media(max-width:600px){ ... }` で小さくする。
- 横幅は `px` の固定値より `max-width`、`vw`、`min(○○px, 92vw)` を使うと画面に収まりやすい。
- 大きな固定サイズの盤面（例：基地防衛の900×600）は、画面幅に合わせて
  `transform: scale()` で全体を縮小すると、スマホでも全体が見えます
  （`games/base-defense/index.html` の `fitGame()` が見本）。

## ✍️ 作者の表示を変えたいとき

作者名は「ラベル（表示）」です。次の場所を直します:

- `README.md` … ゲーム一覧テーブル
- `wiki/index.html` … 各ゲームの見出し
